# 🎮 Conway's Game of Life - Controls Guide

Welcome to the simulation! You can interact with the grid and control the flow of life using the following keys and mouse actions:

---

### 🖱️ Mouse Controls
* **Left Mouse Button (Hold/Click):** Toggle cell state. It allows you to draw patterns on the grid manually by activating cells under the cursor.

### ⌨️ Keyboard Commands

| Key | Action | Description |
| :--- | :--- | :--- |
| **ENTER** | **Start** | Starts the simulation. Life begins to evolve based on Conway's rules. |
| **SPACE** | **Stop** | Pauses the simulation. Useful for observing current patterns or drawing. |
| **R** | **Randomize** | Generates a random initial state across the entire grid. |
| **C** | **Clear** | Wipes the entire grid, removing all living cells. |
| **F** | **Faster** | Increases the simulation speed (FPS) by **+2**. |
| **S** | **Slower** | Decreases the simulation speed (FPS) by **-2** (Minimum: 5 FPS). |

---

### 💡 Tips for Users
* **Monitor Status:** Check the **Window Title** to see if the simulation is currently "Running" or "Stopped".
* **Speed Control:** Use **F** and **S** to adjust the evolution speed. Higher FPS is great for long-term stability tests, while lower FPS is better for analyzing specific cell interactions.
* **Drawing:** You can pause the simulation with **SPACE**, use **C** to clear the board, and then draw your custom "seed" patterns before hitting **ENTER**.
